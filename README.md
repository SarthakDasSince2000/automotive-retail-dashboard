# Automotive Retail Dashboard

A modern web-based Automotive Retail Portal designed to manage dealership operations, analytics, and operational workflows in a single platform.

This project provides a centralized dashboard that connects multiple functional areas such as program tracking, production, supply chain, after-sales, sustainability, and analytics.

---

## 🚀 Features

- 📊 Interactive Dashboard with analytics and operational insights
- 👤 Role-based authentication (Admin & User)
- 📦 Supply Chain management module
- 🏭 Production tracking
- 🧾 Program Tracking system
- 🛠 After Sales management
- 🌱 Sustainability monitoring
- 📈 Analytics and performance metrics
- ⚙ Settings management
- 🌗 Light/Dark theme support
- 📱 Responsive modern UI

## ⚡ Installation & Setup

1. Clone the repository


git clone https://github.com/SarthakDasSince2000/automotive-retail-dashboard.git


2. Navigate to the project folder


cd automotive-retail-dashboard


3. Install dependencies


npm install


4. Run the development server


npm run dev


or


npm start


5. Open in browser


http://localhost:5173

## 🎯 Purpose of the Project

The goal of this project is to demonstrate the development of a **modern enterprise dashboard application** that integrates multiple operational modules of an automotive retail system into a single interface.

This project was built as part of a **technical assessment to showcase frontend development skills using React and modern UI design principles**.
